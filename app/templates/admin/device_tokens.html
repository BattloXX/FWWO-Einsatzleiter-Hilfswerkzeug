Hinweise

  • Die Login-URL enthält einen geheimen Token und ersetzt Benutzername + Passwort.
  • Die URL wird nur beim Erstellen einmalig angezeigt — bitte sofort kopieren oder als Lesezeichen speichern.
  • Widerrufene Token können reaktiviert werden; die URL bleibt dieselbe.
  • Jedes Gerät erhält eine eigene Session und kann unabhängig gesperrt werden.
  • Geräte erscheinen nicht in der normalen Benutzerliste.
  • Wird ein Gerät einem Fahrzeug zugeordnet, erhält es automatisch eine Push-Nachricht, wenn dem Fahrzeug ein Auftrag oder eine Meldung zugewiesen wird.
  • Die Android-App (einsatzleiter.cloud) kann beim ersten Start den QR-Code scannen, um sich dauerhaft einzuloggen.
